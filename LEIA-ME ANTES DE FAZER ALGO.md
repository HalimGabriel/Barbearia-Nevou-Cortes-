## Projeto: Sistema de Agendamento - Nevou Cortes

Este é o repositório do projeto de sistema de agendamento online desenvolvido para a Barbearia Nevou Cortes.

---

## 1. Configuração Inicial do Ambiente

Para executar este projeto localmente, você precisa ter um servidor PHP (como XAMPP, WAMP ou Laragon) instalado.

---

### Pré-requisitos
* Servidor Local: Apache e MySQL/MariaDB ativos.

### Passos de Setup

1. Startar o Servidor: Inicie os módulos Apache e MySQL do seu painel de controle local.
2. Configurar o Banco de Dados:
   - Crie um novo banco de dados no seu ambiente MySQL (via phpMyAdmin, Workbench, etc.).
   - Importar Estrutura: Execute o arquivo .sql contido na pasta do projeto para criar todas as tabelas.
3. Popular Dados: Os dados iniciais de usuários (barbeiros e admin) e serviços serão inseridos automaticamente na primeira vez que você acessar o site.

---

## 2. Acesso ao Projeto

Após o setup, acesse o sistema pelo seu navegador.

* URL de Acesso:
    http://localhost/nome-da-sua-pasta/
